# zm-tree-org

## 介绍
一个简易签名组件，基于vue和canvas。

### 安装
```
npm install zm-sign --save
# or 
yarn add zm-sign
```

### 引入
```
import Vue from 'vue';
import ZmSign from 'zm-sign';
import "zm-sign/lib/zm-sign.css";

Vue.use(ZmSign);
```
